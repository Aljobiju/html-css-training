## Top bar
The top bar provides the service to book an appoinment using a button and a button to have a talk with your prefered doctors.
It also comes with a form that asks you to fill in the details like:
-Full Name  
-Phone Number  
-Date of appoinment  
-Department  
-Doctor Name
